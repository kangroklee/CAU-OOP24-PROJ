# OOP24-PROJ
24-2 객지프 팀플

# Description
This is a Student Information Management System, with full support for inserting/searching/sorting student information. 
This was done as a group project for CAU 2024-2nd semester OOP class.

# Quickstart
## Windows 
Requirements: mingw-w64-ucrt-x86_64-toolchain (includes g++, make)
```
$ make
$ ./s.exe test.txt
```
## macOS
```sh
$ make
$ ./s test.txt
```
## Debian/Ubuntu
```sh
$ make
$ ./s test.txt
```
